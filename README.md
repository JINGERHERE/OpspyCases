
### 介绍
提供一种模块化的openseespy建模思路（具体实现参考对应分支）。

### 模块化建模案例
   - **MomentCurvature**: 截面弯矩-曲率分析
   - **DualColumnBentsTEST**: 双柱式桥墩拟静力试验复现
   - **SelfCenteringBentsTEST**: 自复位双柱式桥墩拟静力试验复现
   - **TallDualColumnBentsFEM**: 双柱式高墩易损性分析（旧版屎山代码，懒得改，无参考意义）

### 建模工具集
   - **opstool**: [文档传送门](https://opstool.readthedocs.io/en/latest/index.html)。
   - **ops_utilities**: [文档传送门](opsu.md)。
